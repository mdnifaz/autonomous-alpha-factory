# SPEC-004 AI Council Specification

## Purpose
Defines collaborative analysis and decision workflow.

## Components
- Coordinator
- Macro Analysis
- Technical Analysis
- Order Flow Analysis
- Funding Analysis
- News Analysis
- Risk Evaluation
- Execution Evaluation

## Workflow
1. Gather evidence
2. Produce assessments
3. Validate findings
4. Aggregate recommendations
5. Allocate portfolio actions
6. Record outcomes

## Outputs
Recommendations, confidence scores, rationale summaries and audit records.

## Controls
Risk evaluation has override authority.
All actions are recorded and reviewable.
